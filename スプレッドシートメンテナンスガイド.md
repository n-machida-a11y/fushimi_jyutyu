# うちわ管理システム　スプレッドシートメンテナンスガイド

---

## 1. スプレッドシートの構成

このシステムは、Google スプレッドシートの **3つのシート** で動作しています。

| シート名 | 役割 |
|---|---|
| `Orders` | 受注データの本体。1行目が「列名（ヘッダー）」、2行目以降がデータ |
| `Config` | プルダウン選択肢の定義。各列の1行目が項目名、2行目以降が選択肢 |
| `ColumnOrder` | 一覧画面の表示列・PDF出力の設定 |

---

## 2. Ordersシート ── 列名の命名規則（最重要）

フォームの表示は **Orders シートの1行目（列名）** だけで制御されます。
列名を変えるだけで、セクション・グループ・表示幅・入力タイプがすべて自動で切り替わります。

### 基本構造

```
セクション名|サブグループ名|サブサブグループ名|表示ラベル
```

`|`（パイプ）または `:`（コロン）で区切ります。

| パーツ数 | 書き方の例 | 結果 |
|---|---|---|
| 1つだけ | `基本` | セクション=基本、ラベル=基本 |
| 2つ | `基本|担当者名` | セクション=基本、ラベル=担当者名 |
| 3つ | `基本|お客様情報|氏名` | セクション=基本、サブグループ=お客様情報、ラベル=氏名 |
| 4つ | `基本|お客様情報|住所|番地` | セクション=基本、サブグループ=お客様情報、サブサブグループ=住所、ラベル=番地 |

---

### セクション（フォームの列）

フォームは **4列** に固定されています。セクション名は以下の4つのみ有効です。

| 列名の先頭 | フォームの列 | 色テーマ |
|---|---|---|
| `基本` | 左端（1列目） | 赤 |
| `印刷` | 2列目 | 青 |
| `うちわ貼り加工` | 3列目 | 黄 |
| `加工` | 右端（4列目） | 緑 |

> **注意：** 上記4つ以外のセクション名を使うと、その列はフォームに表示されません。

---

### サブグループ（フォーム内のブロック）

サブグループ名を指定すると、フォーム内で **枠付きのブロック** にまとめられます。
同じサブグループ名を持つ列は自動的に同じブロック内に並びます。

```
基本|お客様情報|氏名       ← 「お客様情報」ブロックの中に「氏名」
基本|お客様情報|電話番号    ← 同じブロックに「電話番号」も入る
基本|注文内容|商品名        ← 「注文内容」ブロックは別の枠になる
```

---

### 修飾子一覧

列名の**どこにでも**付けられる特殊な記号・キーワードです。複数組み合わせ可能です。

#### ★ （固定表示）
セクションタイトルのすぐ下に常に表示されます（スクロールしても見える位置）。

```
★基本|注文ステータス
基本|★担当者
```

#### ステータス（自動で最上部表示）
列名に「ステータス」という文字が含まれると、セクションタイトルより**さらに上**に自動表示されます。
`★` は不要です。

```
基本|生産管理ステータス   ← セクションタイトルより上に表示される
```

#### [w2]・[w3]・[w4] （表示幅の指定）
フォーム内でのフィールドの横幅を指定します。

| 修飾子 | 幅 | 用途例 |
|---|---|---|
| 指定なし | 全幅（100%） | デフォルト |
| `[w2]` | 半幅（50%） | 2列並べたいとき |
| `[w3]` | 1/3幅（33%） | 3列並べたいとき |
| `[w4]` | 1/4幅（25%） | 4列並べたいとき |

```
基本|サイズ[w2]    ← 半幅で表示
基本|色[w2]        ← 隣に並ぶ
```

#### [half] （サブサブグループの幅を半分に）
サブサブグループ（4階層目）のブロック幅を半分にします。

```
基本|仕様|印刷面[half]|色数
```

#### #番号 （明細行のグループ化）
同じ番号を付けた列は **1行にまとめて** 表示されます。
「名称」「数」「円」を含む列名と組み合わせると、内容・数・単価の3列レイアウトになります。

```
基本|オプション名称#1    ← 内容欄（col-6）
基本|オプション数#1      ← 数欄（col-2）
基本|オプション単価円#1  ← 単価欄（col-4）
```

> 「原価」という文字を含む「円」フィールドは単価欄になりません。

#### @sync(同期先列名) （フィールドの同期）
指定した別の列と値を自動的に同期させます。一方を変更すると他方も変わります。

```
基本|担当者@sync(印刷|担当者)
印刷|担当者
```

---

### 入力タイプの自動判定

列名に特定のキーワードが含まれると、入力タイプが自動で切り替わります。
**Config シートにプルダウン設定がある場合は、キーワードより優先してプルダウンになります。**

| 列名に含まれるキーワード | 入力タイプ |
|---|---|
| `画像` | 画像アップロード（Driveに保存・プレビュー表示） |
| `デザイン` | ファイルアップロード（Driveに保存） |
| `分納・仕分け明細` | ファイルアップロード（Driveに保存） |
| `日` | 日付入力（曜日も自動表示） |
| `数` | 数値入力 |
| `円` | 数値入力 |
| （上記以外）| テキスト入力 |

> **例：** `基本|納品日` → 「日」を含むので日付入力になる
> **例：** `基本|製造数[w2]` → 「数」を含むので数値入力になる

---

### 列名の具体例まとめ

```
ID                              ← システム管理用（変更・削除不可）
作成日時                        ← 自動セット（変更・削除不可）
更新日時                        ← 自動セット（変更・削除不可）
基本|生産管理ステータス          ← セクションタイトル上部に自動表示
★基本|納期日                   ← セクションタイトル直下に固定表示
基本|お客様情報|会社名           ← 「お客様情報」ブロックに入る
基本|お客様情報|担当者名[w2]    ← 同ブロック内、半幅
基本|お客様情報|電話番号[w2]    ← 同ブロック内、半幅（隣に並ぶ）
印刷|仕様|用紙種類              ← 印刷セクションの「仕様」ブロック
印刷|オプション名称#1           ← 明細行グループ1（内容）
印刷|オプション数#1             ← 明細行グループ1（数）
印刷|オプション単価円#1         ← 明細行グループ1（単価）
印刷|デザインデータ             ← 「デザイン」を含むのでファイルアップ
```

## 3. Configシート ── プルダウン選択肢の設定

### 構造

| 列（1行目） | 2行目以降 |
|---|---|
| プルダウンにしたい項目名 | 選択肢を1行ずつ入力 |

例：

| 生産管理ステータス | 用紙種類 |
|---|---|
| 未確認 | コート紙 |
| 確認済 | マット紙 |
| 制作中 | 上質紙 |
| 完了 |  |

### プルダウンが適用される仕組み

Config シートの列名が、Orders シートの列名と **以下の順で照合** されます。

1. Orders列名（修飾子・スペースを除いた文字列）と完全一致
2. 表示ラベル（`|` の最後のパーツ）と完全一致
3. サブサブグループ名と完全一致
4. サブグループ名と完全一致

> **例：** Orders列名が `基本|お客様情報|生産管理ステータス` の場合
> → Config列名が `生産管理ステータス` であればマッチしてプルダウンになる

### 選択肢の追加・変更方法

1. Config シートを開く
2. 対象の列（1行目が項目名）を探す
3. 選択肢を追加・変更・削除する（空白行は自動的に無視される）
4. システムを再読み込みすれば反映される

### 新しいプルダウン項目を追加する方法

1. Config シートの空いている列の1行目に、Orders の表示ラベルと同じ名前を入力
2. 2行目以降に選択肢を入力
3. 完了（コードの変更は不要）

## 4. ColumnOrderシート ── 一覧画面とPDF出力の設定

### 列の構成

| 列名 | 役割 |
|---|---|
| `Home_column` | 一覧画面に表示したい列のOrders列名 |
| `Home_name` | 一覧画面での表示名（省略するとOrders列名そのまま） |
| `Section` | PDF出力を有効にしたいセクション名（基本/印刷/うちわ貼り加工/加工） |
| `PDF_Folder_ID` | PDFの保存先Google DriveフォルダID |
| `PDF_Format_Name` | PDFのテンプレートシート名（同スプレッドシート内のシート名） |

### 一覧画面の表示列を変更する方法

`Home_column` 列に、表示したいOrders列名を1行1列ずつ入力します。
上から順番に左から右へ並びます。

```
Home_column          Home_name
基本|生産管理ステータス  ステータス
基本|お客様情報|会社名   会社名
基本|納期日            納期
```

> `Home_column` が空白の行はスキップされます。

### PDF出力ボタンを追加・設定する方法

1. `Section` 列に対象のセクション名を入力（例：`印刷`）
2. `PDF_Folder_ID` に保存先フォルダのIDを入力
   （フォルダURLの `https://drive.google.com/drive/folders/` の後ろの文字列）
3. `PDF_Format_Name` に同スプレッドシート内のテンプレートシート名を入力

設定すると、そのセクションのフォームに「PDF出力」ボタンが表示されます。

### PDFテンプレートシートの書き方

テンプレートシート内のセルに `{{列名}}` と書くと、そのデータが差し込まれます。

```
{{基本|お客様情報|会社名}}    ← Ordersの「基本|お客様情報|会社名」列の値が入る
{{基本|納期日}}               ← 納期日の値が入る
```

## 5. よくある操作

### ✅ 新しい入力項目を追加したい

1. **Orders シート** の1行目に新しい列名を追加
2. 命名規則に従ってセクション・ラベルを設定
3. プルダウンにしたい場合は **Config シート** にも追加

### ✅ 既存の項目のラベルを変えたい

- Orders シートの1行目の列名の `|` 以降の部分を変更する
- **注意：** 列名（= データのキー）を変えると過去データが紐づかなくなるため、
  列名ごと変更する場合は既存データへの影響を確認すること

### ✅ プルダウンの選択肢を増やしたい

- Config シートの対象列に選択肢を追加するだけでOK

### ✅ 一覧画面に表示する列を変えたい

- ColumnOrder シートの `Home_column` を編集する

### ✅ フィールドを半幅で2列並べたい

- 2つの列名にそれぞれ `[w2]` を付ける

```
基本|サイズ[w2]
基本|色[w2]
```

### ✅ 複数フィールドを同じグループにまとめたい（枠で囲む）

- 同じサブグループ名にする

```
基本|仕様|用紙
基本|仕様|サイズ
基本|仕様|色数
```

---

## 6. 注意事項

- **`ID`・`作成日時`・`更新日時` 列は削除・変更しないこと**（システムが自動管理）
- 列名の変更は過去データに影響する場合があります。変更前にバックアップを推奨
- セクション名は `基本`・`印刷`・`うちわ貼り加工`・`加工` の4つのみ有効
- Config シートの列名とOrders列名のマッチングは、★・[修飾子]・スペースを除いた文字列で比較されます
